# HyChecker
HyChecker is a Python script that allows you to check avaiable usernames for the game [Hytale](https://hytale.com) by [Hypixel Studios](https://x.com/hypixel).

**Download** the latest version of the script [here](https://github.com/y4s/HyChecker/releases/tag/HyChecker).

---

> [!WARNING]
> Dependencies Required:
> 1. [VSCode](https://code.visualstudio.com/) / [Python](https://www.python.org/downloads/)
> 2. Python's Requests Module <br>

### üîç How To Use:
1. Create a new folder 
2. Place `hychecker.py`, `name.txt` & `refresh.py` in the new folder you created
3. Open the folder you created in **VSCode**, your folder should contain the following files: `hychecker.py`, `name.txt` & `refresh.py` 
4. Modify `name.txt` by entering the usernames you're after & save the file <i>(examples are given within the file itself)</i>
5. Run `hychecker.py`

> ‚úÖ If ran correctly, the files: <i>available_names.txt</i>, <i>unavailable_names.txt</i> and <i>name_checked.txt</i> should appear

### üí° Optional:
1. Run `refresh.py` to see if the names you're after are still available

> ‚úÖ When ran, the list of names that are still available will remain in <i>available_names.txt</i>, otherwise moved to <i>unavailable_names.txt</i>
